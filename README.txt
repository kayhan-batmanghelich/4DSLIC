

INTRODUCTION
============

  This code generalizes the SLIC method [2] for 4D volumes; ie 3D images where each voxel is time series or multi-channel.
  To compile this code, you need OpenCV2.2 and cmake-basis [2] and ITK 4+. The input image should be ITK image.
